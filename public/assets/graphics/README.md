Place image or SVG files here and reference them from components or CSS.

Examples:
- Use in JSX: <img src="/assets/graphics/sample-exercise.svg" alt="Exercise" />
- Use in CSS: .hero { background-image: url('/assets/graphics/sample-exercise.svg'); }

Files placed in `public/` are served statically at the site root (e.g. `/assets/graphics/...`).
